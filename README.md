AntiUpdate-Interrupt — это лёгкая утилита для автоматического управления приостановкой обновлений Windows 11 через заданные интервалы времени.

Программа автоматически запускает пользовательский сценарий (win_upd.bat) после истечения заданного количества дней, или часов / минут (!!!Только для теста!!!).

Если вам надоели автообновления Windows, эта программа может помочь:
никаких удалений файлов, редактирования реестра или дополнительных консольных команд, которые могу убить вашу винду (как у меня и произошло) — просто одна безопасная утилита, которую поставил и забыл. Вспоминаешь о ней раз в 35 дней на пару секунд.

⚙️ Возможности

⏱ Интервальное выполнение задач (минуты / часы / дни)

📅 Режим “дни” — проверка только при запуске системы

🔄 Режим “минуты/часы” — фоновый таймер после закрытия окна(!!!Только для теста!!!)

🚀 Автозапуск вместе с Windows

📄 Логирование всех операций

💾 Сохранение настроек в JSON

⚡ Минимальная нагрузка на систему

🧠 Как работает

Days mode:

Проверяет при запуске системы, прошло ли заданное количество дней.

Если да — запускает win_upd.bat (отсрочка обновлений на 35 дней) и завершает работу проги.

Если нет — просто завершает работу проги.

Minutes / Hours mode (!!!Только для теста!!!):

После закрытия приложения запускается фоновый таймер.

По истечении времени выполняется win_upd.bat, после чего программа завершает работу.(!!!Только для теста!!!)

Главный прикол проги в том, что в отличии от кнопки в центре обнов винды, она работает даже при "обязательных" обнов безопасности(у меня получилось, хз как у вас)
